# CDT p2 repository: for Eclipse IDE 2025-09 (4.37.0)

Online Eclipse CDT `12.2.0` p2 repo, including `FreeBSD aarch64/x86_64` artifacts,
built from my [fork](https://github.com/chirontt/eclipse-cdt/tree/cdt_12_2)
of the [Eclipse CDT C/C++ Development Tools](https://github.com/eclipse-cdt/cdt),
to supplement the official Eclipse CDT [12.2.0](https://download.eclipse.org/tools/cdt/releases/12.2/cdt-12.2.0/) p2 repository.

This URL is an Eclipse **software repository**; you must use it in Eclipse ([see how](https://help.eclipse.org/topic/org.eclipse.platform.doc.user/tasks/tasks-127.htm)).
